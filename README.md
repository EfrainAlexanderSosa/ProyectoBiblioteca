# 📚 Sistema de Gestión de Biblioteca

Sistema web desarrollado con Spring Boot para la gestión de bibliotecas.

## 🛠️ Tecnologías
- Java + Spring Boot
- Spring Security (login/roles)
- Spring Data JPA + Hibernate
- MySQL
- Thymeleaf
- Maven

## ✨ Funcionalidades
- Gestión de libros, autores, géneros e idiomas
- Control de ejemplares y estados (disponible, prestado, mantenimiento)
- Gestión de usuarios con autenticación
- Generación de reportes PDF
- Búsqueda de usuarios y ejemplares

## ▶️ Cómo ejecutar
1. Clonar el repositorio
2. Configurar BD en `application.properties`
3. Ejecutar con Spring Boot
4. Acceder a `http://localhost:8080`
